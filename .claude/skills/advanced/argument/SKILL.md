# Create Component Skill

Create a new React component named: **$ARGUMENTS**

#Instructions

Create file: src/components/$ARGUMENTS/$ARGUMENTS.tsx
Create styles: src/components/$ARGUMENTS/$ARGUMENTS.module.css
Create tests: src/components/$ARGUMENTS/$ARGUMENTS.test.tsx
Export from index: src/components/$ARGUMENTS/index.ts
