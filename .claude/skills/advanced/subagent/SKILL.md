# Full Review Skill

Perform a comprehensive review using specialized agents:

#Workflow

@security-agent - Check for vulnerabilities
@performance-agent - Identify bottlenecks
@style-agent - Verify code conventions
@test-agent - Suggest missing tests

#Instructions

Run each agent in parallel, then compile a unified report with:

Critical issues (must fix)
Warnings (should fix)
Suggestions (nice to have)

:::
