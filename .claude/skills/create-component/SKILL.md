# Create Component Skill

Create a new React component named: **$ARGUMENTS**

#Instructions

1. Create file: src/components/$ARGUMENTS/$ARGUMENTS.tsx
2. Create styles: src/components/$ARGUMENTS/$ARGUMENTS.module.css
3. Create tests: src/components/$ARGUMENTS/$ARGUMENTS.test.tsx
4. Export from index: src/components/$ARGUMENTS/index.ts
