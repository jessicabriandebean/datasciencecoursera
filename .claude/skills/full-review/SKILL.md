# Full Review Skill

Perform a comprehensive review using specialized agents:

#Workflow

1. @security-agent - Check for vulnerabilities
2. @performance-agent - Identify bottlenecks
3. @style-agent - Verify code conventions
4. @test-agent - Suggest missing tests

#Instructions

Run each agent in parallel, then compile a unified report with:

Critical issues (must fix)
Warnings (should fix)
Suggestions (nice to have)
